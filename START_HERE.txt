SIMPLE_STT SOURCE OVERHAUL
====================

Architecture:
  AutoHotkey v2 shell -> persistent simple-stt-capture.exe -> disposable simple-stt-infer.exe
  AutoHotkey uses disposable simple-stt-ctl.exe helpers for non-blocking local control.

Read first:
  docs\architecture.md
  docs\configuration.md
  docs\packaging.md
  docs\testing.md
  docs\memory-cleanup-validation.md

Windows development:
  .\scripts\check-prereqs.ps1 -RequireRuntime
  .\scripts\build-release.ps1
  .\scripts\run-dev.ps1 -SkipBuild

One-command distribution build:
  build-distribution.cmd

Outputs:
  artifacts\dist\simple-stt-setup.exe
  artifacts\dist\simple-stt-portable.zip

Static source checks and IPC POC:
  .\scripts\test-static.ps1

The Linux editing environment did not build binaries. Run the documented Windows checks before release.
