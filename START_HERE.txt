UVOX STT - START HERE
=====================

Target: Windows 10+ with an NVIDIA CUDA-capable GPU.

1. Install Rust stable/Cargo and a current NVIDIA driver.
2. Extract the native Parakeet runtime to:

   external\parakeet-runtime\parakeet-windows-cuda

3. Open PowerShell in this folder.
4. Run:

   .\scripts\check-prereqs.ps1
   .\scripts\test-audio.ps1

After the audio test passes:

   .\scripts\run.ps1

Hold CapsLock while speaking. Release CapsLock to transcribe and type.
